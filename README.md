
# TCP Chat 

A server and clinet for chatting using a GUI which uses TCP 

## Features

- Light/dark mode
- save and load chat history
- custom protocol for sending messages  
- private messaging 


## Installation

Install my-project with npm

```bash
  pip install -r requirements.txt
```
